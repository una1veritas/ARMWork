STM32 I2C Communication Peripheral Application Library (CPAL) update history

For more details about STM32 I2C CPAL and examples implementation, please refer to the User manual "UM1029 Communication peripheral application library (CPAL)", available for download from the STMicroelectronics microcontrollers website: www.st.com

V1.2.0 / 17-December-2012

Main Changes

  • All source files: license disclaimer text update and add link to the License file on ST Internet.
  • Add support for STM32F4xx devices (STM324xG-EVAL board) and  STM32L1xx Ultra Low Power High-Density devices (STM32L152D-EVAL board).
  • Miscellaneaous enhancemet of the CPAL drivers
  • Use latest version of STM32F10x, STM32F2xx and STM32L1xx HAL and BSP components (for more details refer to each component's release notes):
    • CMSIS drivers: update directory structure to be compliant with CMSIS V2.1, full API compatibility maintained with previous version
    • Standard Peripherals Drivers: miscellaneous bug fix and enhancement
    • STM32xxx_EVAL board drivers: update drivers architecture and folder organization, full API compatibility maintained vs. V4.4.0
  • All examples: 
    • Project settings updated to be in line with changes made on latest version of the STM32F10x and STM32L1xx HAL and BSP drivers
    • Source code update to be in line with latest version of STM32xxx_EVAL board drivers

V1.0.0 / 31-December-2010

Main Changes

  • First Official version supporting I2C peripheral of STM32F10x and STM32L1xx devices.
  • This release comes with three examples running on STM3210C-EVAL RevB (STM32F10x Connectivity line), STM32100E-EVAL RevB (STM32F10x High-Density Value line) and STM32L152-EVAL RevB  (STM32L1xx Ultra Low Power Medium-Density) evaluation boards and can be easily tailored to any other supported device and development board.
  • For more details about STM32 I2C CPAL and examples implementation, please refer to the User manual "UM1029 Communication peripheral application library (CPAL)", available for download from the STMicroelectronics microcontrollers website: www.st.com/stm32
